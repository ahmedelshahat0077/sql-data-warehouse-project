# Data Warehouse Project

## 📌 Overview

This project demonstrates the implementation of a modern Data Warehouse using the Medallion Architecture (Bronze, Silver, and Gold layers).
The goal is to transform raw data into clean, business-ready datasets for reporting and analytics.
---

## 🏗️ Architecture

<img width="800" height="533" alt="image" src="https://github.com/user-attachments/assets/93f41b7e-8ddd-48ea-9703-796f814c19df" />


## Bronze Layer

- Stores raw data from the source.

- No transformations applied.

## Silver Layer

- Cleans and standardizes the data.

- Removes duplicates and handles missing values.

- Applies business rules.

## Gold Layer

- Creates analytical models.

- Builds fact and dimension tables.

- Provides data ready for reporting and BI tools.
